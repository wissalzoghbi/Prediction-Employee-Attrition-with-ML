# Prediction-Employee-Attrition-with-ML


## Project Overview
Employee attrition (turnover) is a critical challenge for organizations. Predicting which employees are likely to leave helps HR teams take proactive measures to improve retention.  
This project applies **machine learning techniques** to predict employee attrition based on various features such as job role, satisfaction level, and performance metrics.

---

## Tech Stack
- **Python** (Google Colab)
- **Libraries**: pandas, numpy, matplotlib, seaborn, scikit-learn
- **ML Models**: Random Forest Classifier
